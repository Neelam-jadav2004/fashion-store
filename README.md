# Fashion Store

A Full Stack E-Commerce Website built using React, Vite, Django and Razorpay Payment Gateway.

## Features

* User Authentication
* Product Listing
* Product Details
* Shopping Cart
* Wishlist
* Checkout System
* Razorpay Payment Integration
* Order Management
* Order Tracking
* Admin Dashboard
* Responsive Design

## Tech Stack

### Frontend

* React.js
* Vite
* Tailwind CSS
* Axios
* React Router DOM

### Backend

* Django
* Django ORM
* SQLite

### Payment Gateway

* Razorpay

## Installation

### Frontend

```bash
npm install
npm run dev
```

### Backend

```bash
pip install -r requirements.txt
python manage.py runserver
```
